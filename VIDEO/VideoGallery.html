Animated Works

Where'd all the time go? (2019)

An abstract animation created for a junior animation class. The video was created as a fan-made music video for the song Where'd  All the Time Go? by Dr. Dog. I was really interested in experimenting with particle systems and trying to convey meaning/feeling using particles and abstract landscapes. The animation was created using Blender's Cycles and Eevee render engines.

A Strange Encounter (2019)

Animated short created for a sophmore animation class. The goal of assignment was to create a scene where two unlikely characters meet. With that idea in mind, I tried to make a funny interaction between a skeleton in a cave meeting a small dog. I created all the models and textures using Blender, Zbrush and Substance Painter and rendered the animation using Blender's Eevee render engine.

Shape Sounds (2019)

An animated short created in three days for a Junior animation class. My personal goal for the video was to explore non-traditional characters and a world with rules not easily understood. The animation was created and rendered using Blender's Eevee render engine.

Introducing the Hellfire (2018)

A mock-advertisement created on my own time solely for practice. My goal for this project was to create an ad for an 'evil' cellphone while trying to mimic popular advertising for real smartphones. The video features the song I'm Gonna Do My Thing by Royal Deluxe and is rendered using Blender's Cycles render engine.

Robot Love (2017)

An animated short I created in highschool as a personal project. For this project I tried to tell a story without the use of dialog. The animation follows two desktop toys and shows them falling in love. The animation features the song Flowers In Your Hair by The Lumineers and is rendered using Blender's Cycles render engine.
